# Installation

Install Node.js: https://nodejs.org/en/download

In a terminal run these commands:
    npm init -y
    npm install socket.io
    npm install express
    npm install express-session

To run the game:
    In a terminal, run:
        node server
    
    In a browser, run:
        http://localhost:3000
